# Clearing Enhancer

特定のページの要素を書き換えて、日付テキストに曜日表示を追加するChrome拡張機能です。

## 機能

- ページ内の日付テキストを自動検出
- 日付に曜日を追加表示（例：2023/12/25 → 2023/12/25（月））
- 複数の日付形式に対応
- 動的に追加される要素にも対応
- 拡張機能のオン/オフ切り替え

## 対応している日付形式

- `YYYY年MM月DD日` (例: 2023年12月25日)
- `YYYY/MM/DD` (例: 2023/12/25)
- `YYYY-MM-DD` (例: 2023-12-25)
- `MM/DD/YYYY` (例: 12/25/2023)
- `YYYY.MM.DD` (例: 2023.12.25)

## 開発

### 前提条件

- Node.js (16以上)
- npm

### セットアップ

1. 依存関係をインストール
```bash
npm install
```

2. ビルド
```bash
npm run build
```

### Chrome拡張機能として読み込み

1. Chromeで `chrome://extensions/` を開く
2. 「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. プロジェクトの `dist` フォルダを選択

### 開発モード

ファイル変更を監視してリアルタイムでビルドするには：
```bash
npm run dev
```

## 使用方法

1. 拡張機能をインストール後、対象のWebページを開く
2. 自動的に日付に曜日が追加される
3. 拡張機能アイコンをクリックして機能のオン/オフを切り替え可能

## カスタマイズ

`src/content.ts` の `DateEnhancementConfig` を編集することで以下をカスタマイズできます：

- `targetSites`: 対象サイトのリスト
- `dateSelectors`: 日付要素のCSSセレクター
- `datePatterns`: 日付の正規表現パターン
